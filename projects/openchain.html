OpenChain Penguins

OpenChain reached out to Ura to create a series of illustrations to accompany infographics and flyers, which focus on building trust by making open source license compliance simpler and more consistent by using the Open Chain specification.

We created a set of penguin illustrations to bring this message home. OpenChain was so kind to put the illustrations into public domain, so they can be freely reused by anyone for anything.

Further, Ura is also happy to be part of the OpenChain Comformant organizations, fostering further collaboration.
